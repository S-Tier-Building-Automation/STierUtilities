<Note>Provided by **HEIC & MOV** — contract v1.0 — App</Note>

Probe and convert phone media — HEIC/HEIF images and MOV video — via the bundled FFmpeg sidecar.

## Consume it

```js
// 1. Declare it in your tool manifest:
requires: [{ capability: "media.convert", version: "^1.0" }]

// 2. Resolve it from your scoped host and call it:
const mediaConvert = host.use("media.convert.v1");
const result = await mediaConvert.probe(…);
```

## Methods

### `probe(path)`

Returns `Promise<{ kind, ... }>`.

Inspect a file (image vs video, metadata).

### `convert(paths, opts?)`

Returns `Promise<ConvertResult>`.

Convert images→JPEG/PNG and videos→MP4. opts: \{ outputDir?, imageFormat?, overwrite? \}.
