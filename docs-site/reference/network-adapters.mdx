<Note>Provided by **Network Manager** — contract v1.0 — App</Note>

Enumerate and read the live IPv4/DNS state of Windows network adapters.

## Consume it

```js
// 1. Declare it in your tool manifest:
requires: [{ capability: "network.adapters", version: "^1.0" }]

// 2. Resolve it from your scoped host and call it:
const networkAdapters = host.use("network.adapters.v1");
const result = await networkAdapters.list(…);
```

## Methods

### `list()`

Returns `Promise<Adapter[]>`.

All adapters with live metadata.

### `readState(name)`

Returns `Promise<AdapterState>`.

One adapter's current IPv4 address, mask, gateway and DNS.
