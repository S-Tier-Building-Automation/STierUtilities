title: "Capability reference"
description: "Every capability tools on the platform expose, with the contract you build against."
---

Every capability is a **versioned contract**. Declare it in your tool manifest's `requires`, then resolve it from your scoped host with `host.use("<capability>.v<major>")` — you never reach into another tool directly.

## Services

<CardGroup cols={2}>
  <Card title="bacnet.historian" href="/reference/bacnet-historian">

  </Card>
  <Card title="bacnet.read" href="/reference/bacnet-read">

  </Card>
